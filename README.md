# UCB Admissions Shiny App

This project is an interactive Shiny application built in R to explore admission data from UC Berkeley.

## Features

- Dropdown menu to select departments (A to F)  
- Dynamic bar chart showing admission results  
- Reactive updates based on user input  
- Dynamic title that updates with selection  

## Tools Used

- R  
- Shiny  
- Base R datasets  

## Dataset

The dataset used in this project is the **UCBAdmissions** dataset from the R datasets package.

## How to Run

1. Open `app.R` in RStudio  
2. Click **Run App**
